Interactive book proof of concept

This aims to produce an interactive "book". It should be cross-platform, customizable, interactive. It should work on mobile devices such as iPad, iPhone, Android, etc.

Slide the pages or use the scrubber to jump between the chapters. The aA icon changes the font size.
